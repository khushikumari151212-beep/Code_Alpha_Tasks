## 📌My Projects Included

### 🔗 1. URL Shortener
A backend service that converts long URLs into short, easy-to-share links.

#### 🚀 Features
- Generate short URLs
- Redirect to original links
- Unique short code generation
- REST API implementation

#### 📌 Endpoints
- `POST /shorten` → Create short URL  
- `GET /:shortCode` → Redirect  

---

### 🎟️ 2. Event Registration System
A system that allows users to register for events and stores their data securely.

#### 🚀 Features
- Event registration form
- Data validation
- Store user details
- Confirmation on success

#### 📌 Endpoints
- `POST /register` → Register user  
- `GET /registrations` → View all entries  

---

### 🍽️ 3. Restaurant Management System
A backend system to manage restaurant operations like menu and orders.

#### 🚀 Features
- Manage menu items  
- Create and track orders  
- Customer data handling  
- Full CRUD operations  

#### 📌 Endpoints
- `GET /menu` → View menu  
- `POST /order` → Place order  
- `GET /orders` → View orders
